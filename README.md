# Automated-Daily-Operation-Management-System-for-Petrol-Pumps-in-Rural-Maharashtra

The Automated Daily Operation Management System is a mobile application developed to streamline and digitize day-to-day operations at petrol pumps in rural Maharashtra. The system enhances operational accuracy, reduces manual errors, and provides real-time visibility of activities such as employee management, stock updates, and work history tracking.

This app was developed using Flutter for a seamless cross-platform UI and Firebase for backend services including authentication and real-time database operations.
